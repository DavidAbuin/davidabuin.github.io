CS 685 Autonomous Robotics Graduate

George Mason University ·

Fundamentals of autonomous mobile robots: rigid-body transformations and kinematics, sensing and state estimation, motion planning, and feedback control, with hands-on implementation in ROS 2 on both simulated and physical platforms.

Course Objectives

  • Model robot pose and motion using homogeneous transformations and forward/inverse kinematics.
  • Estimate state from noisy sensor data with Bayesian filters (Kalman and particle filters).
  • Plan collision-free paths with graph-based and sampling-based planners (A*, RRT).
  • Design and tune closed-loop controllers, including PID, for trajectory tracking.
  • Build, launch, and debug multi-node ROS 2 systems.

Projects

TurtleBot Waypoint Navigation (ROS 2)

Implemented a ROS 2 navigation stack that fuses odometry and LiDAR to localize a TurtleBot, plans waypoint routes with A* on an occupancy grid, and tracks them with a tuned PID controller. Validated in Gazebo before deployment on hardware.

CS 580 Introduction to Artificial Intelligence Graduate

George Mason University ·

Core techniques of artificial intelligence: uninformed and heuristic search, adversarial search, constraint satisfaction, probabilistic reasoning with Bayesian networks, and an introduction to Markov decision processes and reinforcement learning.

Course Objectives

  • Formulate problems as state-space search and select appropriate algorithms.
  • Apply admissible heuristics and analyze optimality and completeness of A*.
  • Model uncertainty with Bayesian networks and perform inference by sampling.
  • Solve sequential decision problems with value and policy iteration.

Projects

Grid-World Value Iteration Visualizer

Built an interactive visualizer that runs value iteration and Q-learning on configurable grid worlds, animating how utilities and the greedy policy converge as rewards, discount factor, and transition noise change.

CS 310 Data Structures Undergraduate

George Mason University ·

Design, implementation, and analysis of fundamental data structures: dynamic arrays, linked lists, stacks and queues, hash tables, trees, heaps, and graphs, with an emphasis on asymptotic analysis and choosing the right structure for a problem.

Course Objectives

  • Analyze time and space complexity using asymptotic notation.
  • Implement core data structures from scratch and test them rigorously.
  • Select appropriate structures based on access patterns and constraints.
  • Apply graph representations and traversals to practical problems.

Projects

Autocomplete Engine with Tries

Implemented a prefix-tree-backed autocomplete engine over a large word corpus, comparing lookup latency and memory footprint against a sorted array with binary search and a hash-based approach.

CS 262 Introduction to Low-Level Programming Undergraduate

George Mason University ·

Programming in C with a focus on how software meets the machine: the preprocess–compile–assemble–link pipeline, pointers and memory management, arrays and strings, structs, file and process I/O, and Unix development tools.

Course Objectives

  • Write, compile, and debug C programs using gcc, make, gdb, and valgrind.
  • Manage memory explicitly with pointers, malloc/free, and defensive checks.
  • Explain the compilation pipeline from source to executable.
  • Use system calls for file I/O and process control on Unix.
